# TaskFlow

## Descrição

O TaskFlow é um sistema de gerenciamento de tarefas desenvolvido para auxiliar equipes na organização de atividades utilizando conceitos de Engenharia de Software e metodologias ágeis.

## Objetivo

Desenvolver um sistema simples para cadastrar, visualizar, editar e excluir tarefas, permitindo o acompanhamento das atividades de forma organizada.

## Funcionalidades

- Login de usuário
- Cadastro de tarefas
- Listagem de tarefas
- Edição de tarefas
- Exclusão de tarefas

## Metodologia

O projeto utiliza a metodologia ágil Kanban, organizando o fluxo de trabalho através das colunas To Do, In Progress e Done no GitHub Projects.

## Tecnologias

- Python
- Git
- GitHub
- GitHub Actions
- Pytest
